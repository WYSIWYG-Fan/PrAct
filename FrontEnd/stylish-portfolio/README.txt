How to open webpage:

1. Install php-Modules of PostgreSQL
2. Set up XAMPP Server with Apache running
3. Find lines containing "extension=php_pdo_pgsql.dll" and "extension=php_pgsql.dll" in php.ini and make sure they are uncommented
4. Copy source code files into folder C:\xampp\htdocs\PrAct
5. Open webpage in Browser: "http://localhost/PrAct/index.html"
